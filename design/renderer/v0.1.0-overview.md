# v0.1.0 렌더러 개요

## 목표
- CMake 기반 CLI 스켈레톤으로 PPM(P3) 그라디언트 이미지를 생성한다.
- 계약 문서(`design/protocol/contract.md`)에 정의된 해상도/출력 규칙과 동일하게 동작한다.

## 모듈 구성
- `raytracer::Vec3`
  - 3차원 벡터 표현과 덧셈, 스칼라 곱을 제공한다.
  - 향후 기하/색상 표현의 기반 타입으로 확장할 예정이다.
- `raytracer::GenerateGradientImage`
  - `RenderOptions`를 받아 PPM 헤더와 본문을 결정적으로 생성한다.
  - 색상은 `R=round(255*u)`, `G=round(255*(1-v))`, `B=128` 규칙을 사용한다.
- `main`
  - `--width`, `--height`, `--output` CLI 옵션을 해석한다.
  - 옵션이 유효할 때만 `GenerateGradientImage`로 생성한 문자열을 표준 출력 또는 파일로 기록한다.

## 결정성
- RNG 미사용.
- 같은 옵션 입력 시 동일한 문자열을 반환하도록 반올림/클램프 규칙을 고정했다.

## 테스트 커버리지
- 단위: Vec3 덧셈과 스칼라 곱.
- 통합: 4x2 해상도에서 정확한 PPM 문자열 비교.
